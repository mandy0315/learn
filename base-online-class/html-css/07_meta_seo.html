網頁資訊meta

屬性 name/content & http-equiv/content

meta空白元素，是指網頁資訊，不顯示 屬性有兩種name/content;http-equiv/content
name="description"代表搜尋引擎文章與關鍵字能用，區隔
name="robots" content="noindex/index" 代表不使用/使用搜尋引擎功能 content="nofollow/follow" 代表不抓取/抓取頁面連結 兩這能一起使用要加， index開啟通常會造成流量大伺服器負擔
http-eqxiv="author" 網頁作者 http-eqxiv="pragma" 瀏覽快取頁面 content="no-cache" 快取頁面設定 http-eqxiv="expires" 快取頁面日期
